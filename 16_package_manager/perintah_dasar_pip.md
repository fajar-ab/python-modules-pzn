Install package
```sh
pip install nama_package
```

Install versi tertentu
```sh
pip install nama_package=1.2.3
```

Upgrade package
```sh
pip install --upgrade nama_package
```

Uninstall package
```sh
pip uninstall nama_package
```

Lihat package yang terinstall
```sh
pip list
```

Litat info package
```sh
pip show nama_package
```

Install dari requirements.txt
```sh
pip install -r requirements.txt
```
