Buat virtual environment, biasanya penamaan foldernya .venv
```sh
python -m venv nama_folder
```

Aktifkan (Windos)
```sh
my_project_env/Scripts/activate
```

Aktifkan (Mac/Linux)
```sh
source my_project_env/bin/activate
```

Install package dalam environment
```sh
pip install requests colorama
```

Deaktifkan
```sh
deactivate
```
